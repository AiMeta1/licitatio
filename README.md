# Licitatio

Sistema de gestão de licitações públicas integrado ao Supabase.

## Deploy no Vercel

1. Criar repositório no GitHub e fazer upload desta pasta
2. Conectar ao Vercel
3. Configurar variáveis de ambiente (NEXT_PUBLIC_SUPABASE_URL e NEXT_PUBLIC_SUPABASE_ANON_KEY)
4. Deploy

## Variáveis de Ambiente

Veja `.env.local.example` para os valores corretos.
